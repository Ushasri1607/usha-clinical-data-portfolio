## Clinical Data Cleaning

### Objective
To clean and prepare healthcare datasets for analysis by identifying and correcting common data quality issues such as missing values, duplicates, and inconsistent formatting.

---

### Dataset
- Healthcare-related tabular data
- Includes patient demographics and visit-related fields
- Contains missing values, duplicate records, and inconsistent text formats

---

### Tools & Technologies
- SQL
- Microsoft Excel
- Data validation techniques

---

### What I Did
- Identified and removed duplicate patient records
- Standardized text fields (uppercase/lowercase consistency)
- Handled missing values using appropriate cleaning strategies
- Validated data formats for healthcare fields

---

### Key Outcomes
- Improved data accuracy and consistency
- Prepared clean dataset suitable for analytics and reporting
- Reduced risk of errors in downstream healthcare analysis

---

### Skills Demonstrated
- Data cleaning and preprocessing
- Healthcare data quality management
- Attention to detail and validation
